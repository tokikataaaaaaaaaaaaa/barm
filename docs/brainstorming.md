# BARM (Be-a-real-man) Phase 1 Brainstorming Report

**調査日:** 2026-01-02
**判定:** **Conditional Go（条件付きGo）**

---

## Executive Summary

BARMは「男性特化の習慣化アプリ」という**競合不在の市場ポジション**を取れる可能性がある。日本のフィットネスアプリ市場はCAGR 10%以上で成長中であり、参入タイミングは適切。一人開発でも技術的に実現可能。

現行の価格戦略（300円/達成時100円）でも収益性は確保可能だが、達成割引による収益圧迫リスクがある。価格戦略を検討した上で、クローズドβでの検証を経てから本格ローンチを判断すべき。

---

## 1. ビジョン & プロダクト概要

| 項目 | 内容 |
|------|------|
| **ビジョン** | 男としての体づくりと勉強をサポート。仲間の記録を見て頑張れるアプリ |
| **収益モデル** | サブスク月300円、目標達成で次月100円 |
| **機能** | 筋トレ・勉強の種目を自由追加、ミッション自由設定＋アプリ提案 |
| **課題** | モチベーション維持、努力が苦手な若者を救う |
| **ターゲット** | 10代学生、20〜30代社会人男性、人生を変えたい意思がある人 |

## 3. Feasibility Assessment

### 3.1 収益性評価

| 区分 | 規模 |
|------|------|
| TAM（獲得可能最大市場） | 約500億円/年 |
| SAM（アプローチ可能市場） | 約50億円/年 |
| SOM（初年度目標） | 360万円/年（1,000人想定） |

**価格設定の検討事項:**
- 目標達成時100円は、手数料30%を引くと70円
- 達成するほど収益が下がる構造（ただしモチベーション効果あり）
- 競合より安い価格帯で参入障壁を下げる効果

### 3.2 一人開発での実現性

| 観点 | 評価 | 理由 |
|------|------|------|
| 技術的難易度 | ◎ | 標準的なモバイルアプリ技術で実現可能 |
| MVP規模 | ○ | 3-6ヶ月で検証可能 |
| 運用負荷 | △ | コミュニティ機能でモデレーション負荷発生 |
| スケーラビリティ | ○ | BaaSで技術的スケール容易 |

### 3.3 差別化ポイント

| 差別化ポイント | 強度 | 持続性 |
|----------------|------|--------|
| 男性特化 | 高 | 中（先行優位あり） |
| 筋トレ×勉強の統合 | 高 | 高 |
| 仲間の記録が見える | 中 | 低 |
| 達成時の割引 | 中 | 高（ユニーク） |

## 4. Cost Estimation

### 4.1 開発コスト

| 項目 | 見積もり |
|------|----------|
| 開発期間 | 4-6ヶ月（MVP） |
| 開発コスト（自己労働） | 0円 |
| 外部費用 | 約10万円 |

### 4.2 運用コスト

| 項目 | 月額 |
|------|------|
| サーバー（Supabase/Vercel） | 約5,000円 |
| Apple Developer | 約1,000円 |
| Google Play Developer | 約200円 |
| その他 | 約500円 |
| **合計** | **約6,700円/月** |

### 4.3 ROI試算

**前提:**
- 運用コスト: 6,700円 × 12ヶ月 = 約8万円/年
- 初期費用: 10万円
- 広告費: 2万円 × 12ヶ月 = 24万円/年
- **年間コスト合計: 約42万円**

| シナリオ | ユーザー数 | 年間収益 | 年間コスト | ROI |
|----------|-----------|----------|------------|-----|
| 保守的 | 500人 | 84万円 | 42万円 | **+42万円** |
| 現実的 | 1,000人 | 168万円 | 42万円 | **+126万円** |
| 楽観的 | 2,000人 | 336万円 | 50万円 | **+286万円** |

※収益は平均200円/月（達成率50%想定）× 手数料後70%で計算

## 5. Final Recommendation

### 判定: **Conditional Go（条件付きGo）**

### Go条件

**条件1: 価格戦略の確定**

現行案（300円/達成時100円）でも収益性は確保可能。ただし以下も検討の余地あり：

| 案 | 内容 | メリット | デメリット |
|----|------|----------|------------|
| 現行案 | 300円/月、達成時100円 | モチベーション効果、低価格参入 | 達成率高いと収益圧迫 |
| A案 | 480円/月、達成時300円 | 収益性改善 | 価格アップによる心理的障壁 |
| B案 | 300円/月固定 | シンプル、収益安定 | 達成インセンティブなし |

**条件2: MVP検証での成功基準クリア**

| 指標 | 成功基準 |
|------|----------|
| CPI（インストール単価） | 500円以下 |
| Day7継続率 | 30%以上 |
| Day30継続率 | 15%以上 |
| 課金転換率 | 5%以上 |

### 推奨アクションプラン

1. **Phase 1: MVP開発（4-6ヶ月）**
   - iOS先行でMVP開発
   - 最小限機能: ミッション設定、トラッキング、仲間の記録閲覧

2. **Phase 2: クローズドβ（1-2ヶ月）**
   - Twitter/Xで30-50人のβテスター募集
   - 継続率・課金意向のヒアリング

3. **Phase 3: ソフトローンチ（1ヶ月）**
   - ASO最適化してApp Storeリリース
   - 成功基準クリアでスケール

## 6. リスクと懸念事項

### 主要リスク

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|----------|------|
| ユーザー獲得コスト高騰 | 高 | 中 | オーガニック重視 |
| コミュニティ運営負荷 | 中 | 高 | 少人数制、ルール明確化 |
| 継続率の低さ | 高 | 中 | ゲーミフィケーション強化 |
| 達成率高すぎによる収益圧迫 | 中 | 中 | 価格戦略の柔軟な調整 |

### 機会

1. **男性特化市場の空白** - 競合不在の今が参入の好機
2. **ウェルビーイング市場の拡大** - 10兆円市場の成長トレンド
3. **筋トレ×勉強の統合ニーズ** - 包括的な自己改善需要
4. **コミュニティ形成による防御性** - 機能は模倣可能でもコミュニティは模倣困難
